
# Alpha Strike TN Calculator

A quick-reference tool for calculating Target Numbers in **BattleTech: Alpha Strike** combat. Built as a progressive web app (PWA) for seamless use on desktop, mobile, and tablets.

## Features

### 🎯 Calculator Tab
Calculate Target Numbers instantly by adjusting:
- **Attacker Skill** - Your unit's attack skill rating
- **Range** - Short (0), Medium (+2), Long (+4), or Extreme (+6)
- **Target Movement Modifier (TMM)** - Based on target's movement distance
- **Attacker Movement** - Stationary (0), Ran (+1), or Jumped (+2)
- **Cover** - None (0), Partial (+1), or Heavy (+2)
- **Other Modifiers** - Any additional situational effects

The calculator instantly shows your Target Number and a detailed breakdown of all modifiers.

### 📚 Cheat Sheet Tab
Complete Alpha Strike reference tables including:
- **Attacker Movement** - Movement type penalties
- **Target Movement (TMM)** - Distance-based modifiers
- **Range Brackets** - Distance categories and damage values
- **Cover & Terrain** - Environmental effects on attacks
- **Heat & Overheat** - Mech heat penalties
- **Situational Modifiers** - Weather, elevation, and special circumstances
- **Melee Combat** - Melee attack procedures and modifiers
- **Critical Hits** - 2d6 critical damage table
- **Unit Status Conditions** - Prone, shutdown, on fire, etc.

## How to Use

### Basic Workflow
1. **Open the Calculator tab** (default view)
2. **Adjust your inputs:**
   - Set your skill rating
   - Click range buttons or use dropdowns for modifiers
   - Use the cover buttons for quick selection
   - Add any additional "other" modifiers
3. **View the result instantly** - TN and breakdown update in real-time
4. **Reference the Cheat Sheet** - Click the Cheat Sheet tab to check rules mid-game

### Quick Tips
- **Button groups** (Range, Cover) toggle state - click a button to select it
- **Dropdowns** (TMM, Attacker Movement) offer multiple options
- **Real-time calculation** - Results update immediately as you adjust values
- **Easy switching** - Toggle between tabs to reference rules while calculating
- **Works offline** - The app is cached and works without internet

## Offline Support

This app works as a **Progressive Web App (PWA)**, meaning:
- ✅ Works completely offline after the first visit
- ✅ Installed on your device like a native app
- ✅ No internet connection needed once cached
- ✅ Fast loading times
- ✅ No data sent to external servers

